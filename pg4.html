O que Java?

Java é uma linguagem de programação de alto nível, orientada a objetos, desenvolvida pela Sun Microsystems
(agora parte da Oracle) no início dos anos 90. Aqui estão algumas
características e conceitos-chave sobre Java:

1. Portabilidade

Java é projetado para ser portátil, o que significa que o código Java pode ser executado em qualquer plataforma que tenha
a Java Virtual Machine (JVM). Isso é conhecido como "escreva uma vez, execute em qualquer lugar" (WORA).

2. Orientação a Objetos

Java é uma linguagem orientada a objetos, o que significa que ela utiliza conceitos como encapsulamento,
herança e polimorfismo. Isso ajuda a estruturar o código de forma mais modular e reutilizável.

3. Tipagem Estática

Java é uma linguagem de tipagem estática, o que significa que os tipos de variáveis são verificados em tempo de compilação.
Isso ajuda a detectar erros antes que o código seja executado.

4. Gerenciamento de Memória

Java possui um coletor de lixo (garbage collector) que gerencia automaticamente a alocação e liberação de memória,
reduzindo o risco de vazamentos de memória.

5. Bibliotecas e APIs

Java vem com uma vasta biblioteca padrão (Java Standard Library), que fornece uma ampla gama de funcionalidades,
como manipulação de strings, coleções, entrada e saída, rede, e muito mais.

6. Multithreading

Java suporta programação concorrente e multithreading, permitindo que múltiplas threads de execução sejam executadas
simultaneamente, o que é útil para melhorar a performance em aplicações que precisam realizar várias tarefas ao mesmo tempo.

7. Segurança

Java tem um forte modelo de segurança, com recursos como o sandboxing,
que protege o sistema contra códigos maliciosos.

8. Uso e Aplicações

Java é amplamente utilizado no desenvolvimento de aplicações web, aplicações empresariais, aplicativos móveis
(especialmente Android), sistemas embarcados e muito mais.
